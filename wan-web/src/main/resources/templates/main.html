项目信息
wanAdmin是一个简易通用后台管理系统，前端使用笔下光年后台管理模板。
主要技术：
1.核心框架：Spring Boot
2.安全框架：Spring Security
3.模板引擎：Freemarker
4.持久层框架：MyBatis
5.数据库连接池：Druid
插件列表
1.Bootstrap Table表格插件
2.fontIconPicker图标选择插件
3.bootstrap-datepicker日期选择器
4.layer弹窗插件
5.jquery-validate表单验证
6.Bootstrap Notify通知消息
